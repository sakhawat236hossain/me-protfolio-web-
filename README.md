# 🚀 Modern MERN Stack Portfolio

Welcome to my professional portfolio website! This project showcases my journey, skills, and the web development projects I've built using the MERN stack.

## 🌟 About the Developer
I am **MD. Sakhawat Hossain**, a passionate Frontend/MERN Stack Developer. ## 🛠️ Tech Stack & Technologies Used

| Category | Technology |
| :--- | :--- |
| **Framework** | Next.js 16 (App Router), React 19 |
| **Styling** | Tailwind CSS 4, Tailwind Merge, clsx |
| **Animations** | Framer Motion, Typewriter-effect, React Fast Marquee |
| **Icons** | Lucide React, React Icons |
| **Form Handling**| Formspree React |
| **Theming** | Next Themes (Dark/Light Mode) |
| **Navigation** | React Scroll |
| **Others** | SweetAlert2 (Notifications) |

## ✨ Key Features
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
- **Dark/Light Mode:** Seamless theme switching using `next-themes`.
- **Interactive Timeline:** A visual story of my education and programming journey.
- **Service Offerings:** Clearly defined services for clients and recruiters.
- **Smooth Animations:** Premium user experience using Framer Motion 12.
- **Contact Form:** Fully functional integration with Formspree.
- **FAQ Section:** Answering common client queries upfront to save time.
